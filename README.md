## Vercel Deployment

This project is build-tool free and ready for direct static deployment.

### Deploy with Vercel

1. Push this repository to GitHub.
2. Import the repository into Vercel.
3. Vercel will detect it as a static site automatically.
4. No build command is required.
5. Output directory should be left empty.

## Main Files

- `index.html` - main portfolio page
- `assets/css/style.css` - complete styling
- `assets/js/script.js` - interactions and scroll behavior
- `DivyaPrabha.pdf` - downloadable resume

## Notes

- All asset paths are relative for static hosting compatibility.
- The site is optimized for simple deployment on Vercel or any static host.
